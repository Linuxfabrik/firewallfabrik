# Network Discovery

> [!NOTE]
> The network discovery features described in this chapter (SNMP-based object discovery, importing from /etc/hosts files, and importing existing firewall configurations) are not yet available in FirewallFabrik.
>
> To populate your object tree, you can:
>
> - Create objects manually using the GUI (see [05 - Working with Objects](05%20-%20Working%20with%20Objects.md)).
> - Open an existing Firewall Builder file (.fwb) -- FirewallFabrik will import all objects from it. See [17 - Migrating from Firewall Builder](17%20-%20Migrating%20from%20Firewall%20Builder.md) for details.
